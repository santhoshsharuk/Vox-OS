# Python Programming Course - English

This folder contains the English version of the Python Programming course.

## Files

- `course_data.json` - Complete course structure with modules and topics
- `bannerimg.png` - Course banner/thumbnail image (to be added)

## Course Structure

The course contains 10 comprehensive modules:

1. **Python Basics and Getting Started** - Introduction to Python
2. **Variables and Data Types** - Understanding Python data types
3. **Data Structures** - Lists, Tuples, Dictionaries, Sets
4. **Control Flow Structures** - Conditions and loops
5. **Functions** - Writing reusable code
6. **Modules and Packages** - Code organization
7. **File Handling** - Reading and writing files
8. **Exception Handling** - Error management
9. **Object-Oriented Programming** - Classes and objects
10. **Advanced Topics** - List comprehensions, decorators, generators

## Usage

This course data is automatically loaded by the Vox Learner platform when users select the English language option for the Python Programming course.
