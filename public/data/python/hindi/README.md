## Course Structure / पाठ्यक्रम संरचना

पाठ्यक्रम में 10 व्यापक मॉड्यूल हैं:

The course contains 10 comprehensive modules:

1. **Python की मूल बातें और शुरुआत** - Python का परिचय
2. **चर और डेटा प्रकार** - Python डेटा प्रकारों को समझना
3. **डेटा संरचनाएं** - Lists, Tuples, Dictionaries, Sets
4. **नियंत्रण प्रवाह संरचनाएं** - शर्तें और लूप
5. **फ़ंक्शन** - पुन: उपयोग योग्य कोड लिखना
6. **मॉड्यूल और पैकेज** - कोड संगठन
7. **फ़ाइल प्रबंधन** - फ़ाइलों को पढ़ना और लिखना
8. **त्रुटि प्रबंधन** - त्रुटि प्रबंधन
9. **ऑब्जेक्ट-ओरिएंटेड प्रोग्रामिंग** - क्लास और ऑब्जेक्ट
10. **उन्नत विषय** - List comprehensions, decorators, generators
