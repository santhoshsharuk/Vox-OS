## Troubleshooting

### If you don't see the Learning icon:
1. Make sure the application has fully loaded
2. Check the desktop area (top-left of the screen)
3. Try refreshing the application

### If the window doesn't open:
1. Try double-clicking the icon again
2. Check the browser console for errors (F12)
3. Make sure all files are properly loaded

### If videos don't play:
1. Check your internet connection (YouTube videos require internet)
2. Make sure popups are not blocked
3. Some videos may be region-restricted

## Data Persistence

Your progress is automatically saved! You can:
- Close the window
- Refresh the page
- Close the entire application

When you come back, your progress will still be there!

## Quick Tips

1. **Mark topics complete** even if you watch videos elsewhere - this helps track your progress
2. **Check the Progress tab** regularly to see how you're doing
3. **Courses show progress bars** so you can see at a glance what you've completed
4. **All data is local** - stored in your browser's localStorage
